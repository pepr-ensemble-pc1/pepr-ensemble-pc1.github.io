## Thesis Title

[THandgibles] Approche Centrée sur la Main pour l'Interaction Tangible Collaborative

## Abstract

In AR, the tracking of physical objects is important to integrate and use their tangible potential to display or interact with the AR system.

I aim to create an opportunistic system usuable in any situations to track object without training. I focus on the user's hands to gather informations about objects and avoid common errors found in object tracking systems. Users can grasp object to register their position in the system which will then detect when they are manipulated to update the object data.

This lightweight system is designed to be simple to use and compatible with collaborative situations. I aim to develop more potential use cases by integrating useful widgets and potentially computer vision to create useful tools.
